# Bit PROject
Проект Bit PROject решает проблему:
  *поиска информации по видео; 
  *экономит время сотрудникам, не обладая навыками дизайнера;

## Системные требования
Так как проект работает локально то запуск его воспроизводится либо на сервере, либо на компьтере( если локально запускать)

|                    Системные требования                         |
| ------------- |:-----------------------------------------------:|
| ***CPU        |     Intel Core i5 12400 или  AMD Ryzen 5 5600x  |
| ***RAM        |     32GB+                                       | 
| ***GPU        |     RTX 4070 и выше (12 GB VRAM)                |
| ***SSD        |     512 GB и выше                               |
